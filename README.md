# AgroDrones

**MVP de micro-SaaS para registro de aplicações com drones na agricultura**

Sistema web multi-tenant para empresas prestadoras de serviços de pulverização com drones.

**Slogan:** "Cada aplicação registrada. Cada voo comprovado."

## 🚀 Tecnologias

- **Frontend:** React + TypeScript + Vite
- **Estilização:** Tailwind CSS + shadcn/ui
- **Backend:** Lovable Cloud (Supabase)
- **Roteamento:** React Router

## 📋 Funcionalidades

- Dashboard com estatísticas
- Cadastro de Clientes, Drones e Operadores
- Registro de Aplicações conforme MAPA (Portaria 298/2021)
- Relatórios e exportações (CSV/PDF)
- Multi-tenant com isolamento por empresa
- Autenticação e autorização por roles
